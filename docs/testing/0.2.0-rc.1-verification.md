```text

```

A real explicitly configured TLS/HTTP2 client flow produced:


Observed TLS metadata included SNI, ALPN `h2`, and TLS 1.3. The provider reported `certificate_pinning_bypass:false`; this verification does not claim automatic CA installation or pinning bypass.
